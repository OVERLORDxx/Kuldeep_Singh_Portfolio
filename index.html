AWARE ✦ Cryptographic
Blockchain-Inspired Civic Issue Reporting Platform
Code Live Demo

A secure full-stack civic issue reporting platform built to guarantee complaint data integrity using a blockchain-inspired hash chain. Users submit issues with GPS coordinates, converting coordinates to addresses on-the-fly.

  • Designed a SHA-256 hash chaining validation where each report links to the previous entry, preventing database administrators or actors from altering database records.
  • Integrated HTML5 Geolocation API, Leaflet.js map tracking (CartoDB Dark Matter), and the OpenStreetMap Nominatim API to dynamically convert GPS coordinates into address strings.
  • Built custom stateless authentication in Spring Boot using signed HMAC-256 JWT tokens, BCrypt password hashing, and role-based route endpoints.
  • Deployed static React.js frontend on Render, Spring Boot web application on Render, and MySQL database on Aiven Cloud.
React.js Spring Boot MySQL (Aiven) SHA-256 Leaflet.js OpenStreetMap API JWT (HMAC-256) Render

Library Management System ✦ Full-Stack
3-Tier Library Platform & Fine Aggregation Engine
Code Live Demo

A full-stack library management system featuring a responsive React frontend, Express.js backend, and database tracking. Browse, search, borrow & return books with dynamic overdue fine calculations.
